# sales-analysis-using-pivot-tables
Created an interactive sales summary using Microsoft Excel Pivot Tables to analyze and visualize total sales by city. This project demonstrates data aggregation, sorting, and reporting techniques commonly used in business analysis.

## Project Overview
This project demonstrates the use of Microsoft Excel Pivot Tables to analyze sales data and extract meaningful business insights.

## Business Questions Solved

### Task 1: City-wise Sales Analysis
- Identified total sales generated by each city.
- Sorted cities in descending order based on sales.

### Task 2: Sub-Category Performance Analysis
- Calculated the average quantity sold for each sub-category.
- Determined the total discount provided across sub-categories.

### Task 3: Top 5 Profit-Earning States
- Identified the five states contributing the highest profits.

### Task 4: Top 5 Profit-Earning Sub-Categories
- Analyzed product sub-categories to determine the top five based on profit generated.
